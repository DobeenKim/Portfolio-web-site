Hi there, I'm Dobeen Kim!


- Project Overview

  I designed this portfolio in Figma and developed it using Vanilla JS and GSAP for interactions.
  This site includes an About Page for my background, a Projects Section that stays updated via the GitHub API, 
  and a Contact Section. By integrating the API, I ensured that my latest work is automatically displayed in real-time.



- Tech Stacks

  Design & Planning : Figma
  
  Frontend Development
  
    Languages: HTML5, CSS3, JavaScript (ES6+)
    Animation: GSAP, ScrollTrigger, SplitType
    API Integration: GitHub REST API


- Responsive Design

  Range: Fully responsive layout ranging from Mobile (320px) to Desktop (1920px).
